# Pokedex

## Requirements
- Ruby 3.3.2
- node 20.5.0
- SQLite 3

## Setup
- It is necessary to set the following environment variables before running the server: `MISSION_CONTROL_USER` and `MISSION_CONTROL_PASSWORD`
```sh
$ bundle install
$ yarn install
$ bin/rails db:prepare
$ bin/rails assets:precompile
$ bin/rails server
```
